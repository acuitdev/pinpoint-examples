# pinpoint-examples

Examples demonstrating use of the [Acuit Pinpoint API](https://www.acuit.com/docs/dev/index.html) via the [**Acuit.Pinpoint.Server.Client** NuGet package](https://www.nuget.org/packages/Acuit.Pinpoint.Server.Client):

## CustomAcuitPinpointClient

This is a simple example client that connects to Acuit Pinpoint to record test results.

The example targets .NET Framework 4.8 and is written in C#.

## CustomAcuitPinpointClientVB

This is a simple example client that connects to Acuit Pinpoint to record test results.

The example targets .NET Framework 4.8 and is written in Visual Basic.

## CustomAcuitPinpointClientVersion8

This is a simple example client that connects to Acuit Pinpoint to record test results using the new [**SimpleStation**](https://www.acuit.com/docs8/api/Acuit.Pinpoint.Server.Client.SimpleStation.html) helper available in the version 8 preview of the [**Acuit.Pinpoint.Server.Client** NuGet package](https://www.nuget.org/packages/Acuit.Pinpoint.Server.Client), which greatly simplifies implementing custom Acuit Pinpoint stations.

The example targets .NET Framework 4.8.1 and is written in C#.
